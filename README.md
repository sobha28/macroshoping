# macroshoping
 A Java console-based shopping system that uses OOP, user input handling, and regex validation to simulate product browsing and ordering through a menu-driven interface.

A terminal-based shopping system developed using Java and Object-Oriented Programming concepts.
This application allows users to register, browse product categories, and select items using a menu-driven console interface.
